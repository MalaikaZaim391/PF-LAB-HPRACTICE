# Problem: Find Even and Odd Numbers

## Algorithm Steps

- **Step 1:** Start  
- **Step 2:** Input a number `n`  
- **Step 3:** Check the remainder when `n` is divided by 2  
  - If `n % 2 == 0`, then  
    - Output: "`n` is Even"  
  - Else  
    - Output: "`n` is Odd"  
- **Step 4:** End  

## Example
- Input: `n = 7`  
- Process: `7 % 2 = 1` → not equal to 0  
- Output: `7 is Odd`

- Input: `n = 10`  
- Process: `10 % 2 = 0` → equal to 0  
- Output: `10 is Even`
